## 📃 TIL Rule

</div>

📌기본적인 Rule은 변경되지 않지만 언제든 추가될 수 있습니다.
* 일일 commit을 목표로 합니다.
* 목표로 하되, commit을 채우려고 억지로 하지 않습니다.
* 하루에 commit을 못한 경우 그 하루의 commit은 넘어가야합니다.
* 각 달마다, 일마다 날짜를 기재하여 commit 합니다.
* Reference 명확히 기재합니다.
* 편하게 쓰되, 나중에 자신이, 남들이 보기에 쉽게 알아볼 수 있어야 합니다.
